# Ansible Collections

Workflows for testing Ansible collections and releasing them to [Galaxy](https://galaxy.ansible.com).

| Workflow                                                                          | Description                                           |
| --------------------------------------------------------------------------------- | ----------------------------------------------------- |
| [Release](https://radiorabe.github.io/actions/workflows/ansible/release/index.md) | Build and publish an Ansible collection to Galaxy     |
| [Test](https://radiorabe.github.io/actions/workflows/ansible/test/index.md)       | Lint an Ansible collection with ansible-lint and ruff |
